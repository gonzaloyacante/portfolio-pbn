# Code of Conduct

## Our Commitment

This repository is a professional portfolio showcase. Interactions must remain respectful, professional, and focused on technical quality.

## Expected Behavior

- Be respectful and constructive
- Focus feedback on ideas and code, not people
- Assume good intent
- Report security concerns privately

## Unacceptable Behavior

- Harassment, insults, threats, or personal attacks
- Publishing private information
- Sharing credentials, tokens, or sensitive operational details
- Bad-faith or abusive interactions

## Enforcement

The project owner may moderate, hide, lock, or remove interactions that violate this code.

## Scope

This code applies to all repository interactions (issues, pull requests, discussions, and comments).
